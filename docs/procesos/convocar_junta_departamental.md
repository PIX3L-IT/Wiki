# Agendación de un tema para junta departamental

## Propósito

Agendar un tema a discutir con el departamento

## Miembros y/o Roles Involucrados

- **Rol**: Secretario.

## Entradas

1. **Requerimiento 1** Tener un tema a tratar con el departamento.
